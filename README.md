## dy_lottery_helper项目的可多开版本的服务端
此项目将查找直播间的算法单独用一个程序处理，尽可能避免单IP受到影响

## 2026/04/03
## 版本1.7.0
新增逻辑:如果检测到账号处于被风控状态(此时会无法参与福袋，登录此账号也将无法获取到福袋信息)，用于查找直播间的浏览器会自动切换到未登录状态(这是另一个独立的浏览器，不要登录！)

## 2026/03/29
## 版本1.6.0
现在用于查找直播间的浏览器会定时(每两小时)切换账号，以确保能比较稳定地调用接口

**两个账号在不同的浏览器上，请确保这两个浏览器你都扫码登录了账号**

## 2026/03/24
## 版本1.5.0
1.新增了方法’通过全部入口查找直播间’，该方法和方法’通过类别查找直播间’交替使用(20:00-24:00这段时间只会使用方法’通过类别查找直播间’)，以提高查找有效直播间的效率

2.去除了从record*中查找直播间的方法，因为该方法会触发风控机制(会导致无法获取福袋信息)

3.延长了用于查找直播间的浏览器的页面的自动刷新时间
